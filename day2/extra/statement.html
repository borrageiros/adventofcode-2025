El dependiente descubre rápidamente que todavía hay IDs inválidos en los rangos de tu lista. ¿Quizás el joven Elfo estaba haciendo otros patrones tontos también?

Ahora, un ID es inválido si está compuesto únicamente por alguna secuencia de dígitos repetida al menos dos veces. Así, 12341234 (1234 dos veces), 123123123 (123 tres veces), 1212121212 (12 cinco veces) y 1111111 (1 siete veces) son todos IDs inválidos.

Del mismo ejemplo de antes:

  • 11-22 sigue teniendo dos IDs inválidos: 11 y 22.
  • 95-115 ahora tiene dos IDs inválidos: 99 y 111.
  • 998-1012 ahora tiene dos IDs inválidos: 999 y 1010.
  • 1188511880-1188511890 sigue teniendo un ID inválido: 1188511885.
  • 222220-222224 sigue teniendo un ID inválido: 222222.
  • 1698522-1698528 sigue sin contener IDs inválidos.
  • 446443-446449 sigue teniendo un ID inválido: 446446.
  • 38593856-38593862 sigue teniendo un ID inválido: 38593859.
  • 565653-565659 ahora tiene un ID inválido: 565656.
  • 824824821-824824827 ahora tiene un ID inválido: 824824824.
  • 2121212118-2121212124 ahora tiene un ID inválido: 2121212121.

Sumando todos los IDs inválidos de este ejemplo se obtiene 4174379265.
